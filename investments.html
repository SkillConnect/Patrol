    Date
    This is the date that the Financial Institution has provided along with the other transaction details. In most cases, this is the date on which the transaction was performed. In some cases, this could be the date when the transaction actually got Posted.
    Amount
    This is the Amount for the transaction. A negative (-) sign indicates that this was a Debit transaction. A positive (+) sign indicates that this was a Credit transaction.
    Merchant
    This is the name of the Merchant that the Financial Insitution has shared. Sometimes, there is additional meta data (description) shared in the Merchant name. If you want to clean up the Merchant name, please click on the Edit icon from the Action column.
    Category
    The Category for a transaction is assigned to create a meaningful grouping of transactions. If a Category has been incorrectly assigned to a transaction and you want to change it or if you want to add a new category, please click on the Edit icon from the Action column.
    Institution
    This is the Financial Institution associated with the transaction. This field also has the Account Number in brackets to indicate the account associated with the transaction.
    Tags
    You can use this field to add custom groupins for a transaction. For example, if you want to track your Expenses for your vacation, you can add specific tags to the related transactions. You can add one or multiple tags. Once you type in the tag name, press Enter to save the Tag. You can also use the Reports section from the top navigation bar to generate report specific to your Tags.
    Type
    The Type field indicated the Transaction Type. Based on the other details of the transaction, the Type is auto assigned. If a Type has been incorrectly assigned to a transaction and you want to change it, please click on the Edit icon from the Action column.
    REVIEWED?
    By default, all transactions are marked as NOT REVIEWED. The REVIEWED status indicates that you have REVIEWED all the details of the transaction. You can mark a transaction as 'REVIEWED' by clicking on the tick mark option from the Action column. You can also change status by clicking on the actual NOT REVIEWED text link which will open a pop-up that will allow you to change status. From this pop-up, you can also mark a transaction as REVIEW LATER.
    Action
    This column has three options: TICK MARK icon, EDIT icon and DELETE icon. TICK MARK option is to mark a transaction as REVIEWED. EDIT option is to modify details of the transaction. DELETE option is to delete the transaction - once deleted, a transaction is permanently deleted from the system and it will not be retrieved.
